# zimaboard-truenas-ce-nas
Low-power 16TB NAS built on Zimaboard 2 with TrueNAS CE

## Goals
- Build a compact NAS
- Create a reproducible setup guide
- Document the entire process with issues/what I learned

## Hardware
- Zimaboard 2
   - 16GB RAM
   - Intel N150
   - 64GB Onboard Flash (Unused - Unreliable)
   - 500GB USB SATA SSD
   - 2x8TB WD HDD
 
## Storage
- Mirror Pool
- 8TB Usable
- ZFS

## Services
- Immich
- Syncthing
- Media Storage
- Time Machine backups
